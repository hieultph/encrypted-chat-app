## How to Use This Project

Follow these instructions to get started:

1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Run the project:
    ```sh
    npm run dev
    ```

That's it! Now you can customize and tweak the project as needed.
